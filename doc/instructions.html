Setting up Raspberry Pi 3

  • Install Raspbian OS - RASPBIAN JESSIE LITE, Kernel version: 4.4 on an SD Card that has at least 32 Gb of memory from here
  • Insert the SD card with the image onto the Raspberry Pi.
  • Connect the Raspberry Pi to the Internet.
  • Open Terminal
    1. Run:$ sudo apt-get update
    2. Run:$ sudo apt-get upgrade
    3. Run:$ sudo raspi-config
  • Press 1: To Change the password
  • Press 5: Interface Options
  • Press P2: Enable SSH
  • Press 5: Interface Options
  • Press P5: Enable I2C
  • Press 4: Localization Options
  • Press I2: Change Time Zone
  • Select: US, Eastern
  • Press 7: Advanced Options
  • Press A1: Expand Filesystem
  • Select Yes when prompted
  • Select Finish
  • Select Yes to reboot
